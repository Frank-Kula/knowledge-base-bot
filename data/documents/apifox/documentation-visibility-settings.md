# 文档可见性设置


“文档可见性设置” 可以方便管理者灵活控制项目资源的访问权限。


可以将目录、API 文档、接口用例、Markdown 文档和数据模型设置为 “共享” 或 “内部”。

- 共享：默认情况下，资源的可见性为“共享”。团队成员可以查看、修改并对外分享上述资源。
- 内部：资源可见性设置为“内部”时，团队成员依然可以查看并在线分享，但无法对外公开发布。

:::tip[]
Apifox 版本号需`>=2.6.16` 才能进行设置。
:::
