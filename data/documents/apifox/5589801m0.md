# 车辆（Vehicle）

生成与车辆相关的条目的模块。

<br />

**概述**

大多数方法都与汽车有关：`{{$vehicle.vehicle}}` 名称由汽车 `{{$vehicle.manufacturer}}` 和 `{{$vehicle.model}}` 组成。你还可以生成 `{{$vehicle.fuel}}`、`{{$vehicle.type}}` 和 `{{$vehicle.color}}`，以及典型的汽车登记 ID `{{$vehicle.vin}}` 和 `{{$vehicle.vrm}}`。

如果你更喜欢两轮车，则可以生成 `{{$vehicle.bicycle}}` 类型。



## bicycle

返回一种自行车类型。

**返回**: 字符串

**示例**

```js
{{$vehicle.bicycle}}  // '城市自行车'
```


## color

返回车辆颜色。

**返回**: 字符串

**示例**

```js
{{$vehicle.color}}  // '蓝绿色'
```


## fuel

返回一种燃料类型。

**返回**: 字符串

**示例**

```js
{{$vehicle.fuel}}  // '汽油'
```


## manufacturer

返回制造商名称。

**返回**: 字符串

**示例**

```js
{{$vehicle.manufacturer}}  // '吉普'
```


## model

返回车辆型号。

**返回**: 字符串

**示例**

```js
{{$vehicle.model}}  // '普锐斯'
```


## type

返回车辆类型。

**返回**: 字符串

**示例**

```js
{{$vehicle.type}}  // '双排座皮卡'
```


## vehicle

返回一个随机车辆。

**返回**: 字符串

**示例**

```js
{{$vehicle.vehicle}}  // '宝马 PT 漫步者'
```


## vin

返回车辆识别码 (VIN)。

**返回**: 字符串

**示例**

```js
{{$vehicle.vin}}  // 'E2AD154L2TK700685'
```

## vrm

返回车辆登记号码（车辆登记标志 - VRM）

**返回**: 字符串

**示例**

```js
{{$vehicle.vrm}}  // 'OT08HEE'
```
