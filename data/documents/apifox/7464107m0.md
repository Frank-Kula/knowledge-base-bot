# 接口参数值如何设置为 null

可以使用 `{{变量名|default('null', true)}}`来设置参数值。

这样，当变量的值为空时，请求会发送 `null`；当值不为空时，请求会发送实际的变量值。

例如有一个环境变量 `name`，在请求参数中写作：

```json
{
  "name": {{name|default('null', true)}}
}
```

如果环境里的 name 有值，例如 "Tom"，发送请求时为：


```json
{
  "name": "Tom"
}
```

如果环境里的 name 没有值，请求时会发送：

```json
{
  "name": null
}
```
