# Apifox 免费版常见问题（FAQ）：关于人数、团队与协作限制的说明

近期我们注意到，一些竞品黑稿文章里出现了关于 Apifox 免费版 “限制协作人数” “免费仅限少数人使用” 等不实说法，上述说法与 Apifox 实际产品规则 **严重不符**。

为避免误导用户，现就 Apifox 免费版在成员数量、项目规模及团队协作相关功能上的规则作出统一、明确说明。

## Q1：Apifox 是免费的吗？

是的。

**Apifox 提供永久免费版本**，用户可以长期免费使用，无试用期限制。


## Q2：Apifox 免费版是否限制使用人数或协作人数？

**不限制。**

Apifox 免费版 **不限制项目成员数量，也不限制协作人数**。

任意数量的成员都可以加入同一个项目，共同进行 API 设计、文档维护、接口调试和测试协作。


## Q3：使用 Apifox 免费版，是否需要为每个成员单独付费？

**不需要。**

Apifox 免费版 **不按人头收费**。

所有加入项目的成员均可免费使用核心功能，无需因成员数量增加而额外付费。



## Q4：一个项目中最多可以加入多少人？

**没有上限。**

Apifox 免费版 **不对单个项目的成员数量设上限**，任意人数均可免费加入同一个项目协作。



## Q5：是否可以创建多个项目？项目数量是否有限制？

**不限制。**

Apifox 免费版 **不限制项目创建数量**，可根据需要创建任意数量的项目，且每个项目均支持多人协作。



## Q6：是否可以创建多个团队？团队数量是否有限制？

**不限制。**

Apifox 免费版 **不限制团队创建数量**。

用户可以创建并管理多个团队，每个团队下均可包含多个项目和成员。


## Q7：是否可以邀请外部成员或合作伙伴加入项目？

可以。

Apifox 免费版支持通过 **账号或邮箱邀请外部成员**（如合作伙伴、跨团队成员）加入项目协作，

**不限制受邀人数，也无需为外部成员额外付费**。



## Q8：多人是否可以同时编辑同一个 API 或文档？

可以。

Apifox 免费版支持 **多人实时协同编辑接口与 API 文档**，多人同时操作时内容会自动同步，**不限制同时协作的人数**。



## Q9：团队成员是否都可以进行接口调试和测试？

是的。

在同一项目内：

* 所有成员均可进行接口调试
* 所有成员均可创建、运行和维护自动化测试
* 测试用例与测试集合可在团队内复用

以上行为 **均不按人数限制或收费**。



## Q10：环境配置、Mock 接口是否可以在团队内共享？

可以。

Apifox 免费版支持：

* 创建多个环境配置
* 将环境变量设置为团队共享，成员自动同步
* 创建并共享 Mock 接口供团队成员使用

这些能力 **不因成员数量增加而受限**。



## Q11：Apifox 免费版是否存在“仅限 1 人使用”的情况？

**不存在。**

Apifox 免费版在产品设计上 **不限制使用人数、协作人数、项目人数**，也不存在“免费仅限 1 人 / 1 项目 / 1 团队”的使用限制。



## Q12：那 Apifox 不同版本之间的差异主要体现在哪里？

不同版本的差异 **不在于人数限制**，而主要体现在：

* 高级权限与更细粒度的角色管理
* 企业级管理与审计能力
* 部分高级能力与配额策略

这些属于 **进阶与企业级能力**，而非基础协作门槛。



## 总结

**Apifox 免费版在以下“规模维度”上均不设限制：**

* 不限制成员数量
* 不限制协作人数
* 不限制项目数量
* 不限制团队数量
* 不按人数、项目或团队规模收费

Apifox 希望通过免费版，让个人开发者与团队无需担心人数成本，即可顺畅协作与使用。**如果是需要更为高级的功能或私有化部署，那么才需要 [升级版本](https://apifox.com/pricing)。**
