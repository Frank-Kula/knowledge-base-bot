# 环境变量配置

**来源**: https://docs.apifox.com/env-management

## 什么是环境变量

环境变量用于在不同的运行环境（开发、测试、生产）之间切换配置。

## 创建环境

### 步骤

1. 在项目中点击"环境管理"
2. 点击"新建环境"
3. 输入环境名称（如：开发环境、测试环境、生产环境）
4. 添加变量

## 常用变量

### base_url

接口的基础 URL。

**开发环境**：
```
https://dev-api.example.com
```

**生产环境**：
```
https://api.example.com
```

### token

认证令牌。

```
Bearer your-token-here
```

## 使用环境变量

在接口中使用 `{{base_url}}` 引用变量。

**示例**：
```
{{base_url}}/users
```

会自动替换为对应环境的 URL。

## 切换环境

在发送请求前，选择要使用的环境即可。

## 常见问题

### Q: 变量不生效？

A: 检查变量名是否正确，确保没有拼写错误。

### Q: 如何在不同环境之间快速切换？

A: 在顶部环境栏下拉选择即可。
