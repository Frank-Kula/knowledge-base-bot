# 如何将 Apifox 项目导入到离线空间里？

1. **导出项目**：在Apifox网页端/客户端中，进入 **项目设置 > 导出**，选择 **OpenAPI Spec** 格式，导出文件。  
2. **导入离线空间**：打开离线空间，点击 **导入集合 > 选择 OpenAPI(Swagger)文件**，选中已导出的文件完成导入。  


:::caution[]
 **注意事项**：  
- **功能限制**：离线空间仅支持 API 调试，不包含文档、Mock、自动化测试等功能，适合个人 / 小团队快速调试或无网络 / 高隐私场景。  
- **数据存储**：离线空间数据以文件形式存储于本地，**不支持云端同步**，需手动通过文件管理（如Git）实现协作。  
- **导入规则**：若原 Apifox 项目中存在相同接口（`method`和`path`一致），导出时会自动去重，此时导出再导入离线空间属于有损导入。  
:::
