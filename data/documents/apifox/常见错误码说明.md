# 常见错误码说明

**来源**: https://docs.apifox.com/error-codes

## HTTP 状态码

### 2xx 成功

- **200 OK**: 请求成功
- **201 Created**: 资源创建成功
- **204 No Content**: 请求成功，无返回内容

### 4xx 客户端错误

- **400 Bad Request**: 请求参数错误
- **401 Unauthorized**: 未授权，需要登录
- **403 Forbidden**: 禁止访问
- **404 Not Found**: 资源不存在
- **422 Unprocessable Entity**: 请求格式正确但语义错误

### 5xx 服务器错误

- **500 Internal Server Error**: 服务器内部错误
- **502 Bad Gateway**: 网关错误
- **503 Service Unavailable**: 服务不可用

## Apifox 特有错误

### 认证错误

**错误**: API Key 无效

**解决**:
- 检查 API Key 是否正确
- 确认 API Key 已激活
- 联系管理员

### 限流错误

**错误**: 请求过于频繁

**解决**:
- 降低请求频率
- 升级套餐
- 联系技术支持

## 错误处理建议

### 1. 记录完整错误信息

包括：
- 错误码
- 错误消息
- 请求参数
- 响应内容

### 2. 查看日志

在 Apifox 中查看 Console 日志：
- 点击 Console 标签
- 查看详细错误信息
- 下载 HAR 文件分析

### 3. 联系支持

如果无法解决：
- 提供完整错误信息
- 提供复现步骤
- 提供项目和接口信息
