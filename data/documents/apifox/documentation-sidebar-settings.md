# 文档左侧目录设置


Apifox 为 API 文档提供了左侧目录设置功能，可以根据你的需要，设置成你想要的样式和布局。


:::tip[]
Apifox 版本号需 `>=2.6.6` 才能进行设置。
:::

## 设置入口

当你想要特殊设置某个目录样式的时候，在 Apifox 中，点击你选择的目录进行设置。


<Background>

![image.png](https://api.apifox.com/api/v1/projects/5097254/resources/484852/image-preview)
</Background>


## 目录树展示方式

**普通目录：** 作为默认选项，目录是以文件夹的形式展示。点击目录时，可以展开/收缩目录。
  
<Background>

![image.png](https://api.apifox.com/api/v1/projects/5097254/resources/484853/image-preview)
</Background>


**模块：** 以标题样式显示并自动展开该目录下的一级子标题。


<Background>

![image.png](https://api.apifox.com/api/v1/projects/5097254/resources/484854/image-preview)
</Background>


**跳转到子级目录：** 当某个目录下的内容非常丰富时，可以选择这个样式。点击目录时，会聚焦当前目录内容。


<Background>

![doc-directory-config-06.gif](https://api.apifox.com/api/v1/projects/5097254/resources/484856/image-preview)
</Background>


**默认展开目录：**


![image.png](https://api.apifox.com/api/v1/projects/5097254/resources/484858/image-preview)

- 默认选型为 “全局自动”，会根据所有目录内的内容数量，自动展开/收起目录。使阅读者打开文档时，左侧目录树是丰富的。
  
  
    <Background>

    ![image.png](https://api.apifox.com/api/v1/projects/5097254/resources/484859/image-preview)
    </Background>


- 你也可以选择 “手动设置”，根据你的需要，设置某个目录默认展开/收起。

  
    <Background>

    ![image.png](https://api.apifox.com/api/v1/projects/5097254/resources/484860/image-preview)
    </Background>



## 目录内容展示设置

**自定义内容：** 作为默认选项，点击目录时，会展示目录的文字内容，可作为一个模块的说明。


<Background>

![image.png](https://api.apifox.com/api/v1/projects/5097254/resources/484861/image-preview)
</Background>


**展示子级内容列表：** 点击目录时，会展示该目录下的内容索引，帮助读者快速了解该目录下的内容。


<Background>

![image.png](https://api.apifox.com/api/v1/projects/5097254/resources/484862/image-preview)
</Background>


**不展示：** 点击目录时，仅作为文件夹展开/收起，本身没有内容显示。
