# API Hub


Apifox 提供了一个 API Hub，里面包含了数百个主流 API。在这里，你可以阅读它们的 API 文档，或者直接发送 API 请求。

</Background>

## 浏览 API

在 API Hub 中，点击某个 API 的 “**在线文档**” 就可以查看它的 API 文档。

你也可以点击 “**用 Apifox 打开**” 直接进入该 API 的编辑和调试界面。

如果你想将这个 API 复制到自己的账户下使用，需要点击左侧的 “**克隆项目**” 按钮，将它复制到你自己的团队中。

## 贡献 API Hub

当你发布 API 文档时，可以将它贡献到 API Hub 中。

在 「**分享文档** -> **发布设置**」 中，点击发布时启用 “发布到 API Hub” 选项。Apifox 的运营团队会审核这份文档，决定是否将其加入 API Hub。
