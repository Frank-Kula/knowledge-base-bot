# 接口文档生成

**来源**: https://docs.apifox.com/api-docs

## 功能概述

Apifox 支持一键生成美观的 API 文档。

## 生成文档

### 步骤

1. 完成接口设计
2. 点击"分享文档"
3. 选择文档格式（Markdown、HTML、Word 等）
4. 生成并下载

## 文档内容

### 包含信息

- 接口名称
- 请求方法和 URL
- 请求参数
- 响应示例
- 错误码说明
- 注意事项

## 在线分享

### 公开链接

生成的文档可以生成公开链接，分享给团队成员。

### 权限控制

- 设置访问密码
- 限制访问域名
- 设置过期时间

## 常见问题

### Q: 文档格式不对？

A: 检查接口定义是否完整，确保所有必需字段都已填写。

### Q: 如何批量生成？

A: Apifox 支持一键生成整个项目的文档，无需逐个接口生成。
