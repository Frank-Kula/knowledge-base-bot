# 如何在 Markdown 文档中插入项目内接口/文档/数据模型/目录？

Apifox的 Markdown 文档支持便捷地引用项目相关资源，增强文档的关联性与实用性。
#### 插入接口、文档或目录
点击`插入项目内接口/文档`按钮，之后便会弹出选择窗口，你可以从中挑选需要插入的接口、文档，选中后即可插入到文档指定位置。
<Background>
![image.png](https://api.apifox.com/api/v1/projects/5097254/resources/526080/image-preview)
</Background>
如果是插入的资源是接口目录，直接点击目录右侧的`插入`。
<Background>
![image.png](https://api.apifox.com/api/v1/projects/5097254/resources/526082/image-preview)
</Background>
#### 插入数据模型
若想插入本项目的数据模型，可依次点击`插入 - Apifox组件 - 数据模型` ，完成数据模型的添加。
<Background>
![image.png](https://api.apifox.com/api/v1/projects/5097254/resources/526083/image-preview)
 </Background>
