# 使用 Runner 运行测试场景报错，报告提示"unsafe legacy renegotiation disabled"?

当使用 Runner 运行自动化测试场景查看测试报告时，某步骤出现以下报错
```js
error:0A000152:SSL routines:final renegotiate:unsafe legacy renegotiation disabled
```
可能原因是服务器的 SSL 协议过低，Runner 目前不兼容这种请求。可在 docker 运行此命令解决这个问题。
```js
docker pull registry.cn-hangzhou.aliyuncs.com/apifox/self-hosted-general-runner:debian
```
