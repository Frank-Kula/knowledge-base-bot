# 公司（Company）

生成公司相关条目的模块。

<br />

**概述**

要生成随机公司名称，请使用 `name()`。这在许多语言环境中都是本地化的。

要生成充满行话的公司标语和流行语，请使用 `catchPhrase()` 或 `buzzPhrase()`。

<br />


**相关模块**

对于产品和商业，请使用 `commerce` 类型。
对于与金融相关的条目，请使用 `finance` 类型。



## buzzAdjective
返回一个随机的流行形容词，可用于演示经理查看的数据。

**返回**: 字符串

**示例**

```js
{{$company.buzzAdjective}}  // '后端'
```



## buzzNoun
返回一个随机的流行名词，可用于演示经理查看的数据。

**返回**: 字符串

**示例**

```js
{{$company.buzzNoun}}  // '终身价值'
```



## buzzPhrase
生成一个随机的流行短语，可用于演示经理查看的数据。


**返回**: 字符串

**示例**

```js
{{$company.buzzPhrase}}  // '实施直观的指标'
```



## buzzVerb
返回一个随机的流行动词，可用于演示经理查看的数据。

**返回**: 字符串

**示例**

```js
{{$company.buzzVerb}}  // '参与'
```



## catchPhrase
生成一个随机的标语，可以显示给最终用户。

**返回**: 字符串

**示例**

```js
{{$company.catchPhrase}}  // '扩展的专用核心'
```



## catchPhraseAdjective
返回一个随机的标语形容词，可以显示给最终用户。

**返回**: 字符串

**示例**

```js
{{$company.catchPhraseAdjective}}  // '三缓冲'
```



## catchPhraseDescriptor
返回一个随机的标语描述符，可以显示给最终用户。

**返回**: 字符串

**示例**

```js
{{$company.catchPhraseDescriptor}}  // '非易失性'
```


**返回**: 字符串

**示例**

```js
{{$company.catchPhraseAdjective}}  // '三缓冲'
```



## catchPhraseNoun
返回一个随机的标语名词，可以显示给最终用户。

**返回**: 字符串

**示例**

```js
{{$company.catchPhraseNoun}}  // '安装'
```



## name

返回一个随机的标语名词，可以显示给最终用户。

**返回**: 字符串

**示例**

```js
{{$company.name}}  // 'Torphy - Gleason'
```
