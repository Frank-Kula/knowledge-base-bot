# 如果一个字符串属性有多个枚举值，而且在不同地方都用到了，怎么才能保证这个枚举在整个文档里都保持一致？

你可以把这个属性定义成一个单独的[数据模型](https://docs.apifox.com/create-data-schema.md)，里面就一个属性。这样你就能在 API 文档的不同地方统一引用它了。例如：
