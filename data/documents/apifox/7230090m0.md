# Git 仓库连接

> 请确保 Apifox 版本 `≥2.7.29`

Apifox 支持将项目中各模块的 **OpenAPI / Swagger 文件**自动备份到 Git 仓库，目前支持 GitHub 和 GitLab。

## 功能说明

* 支持自动备份 Apifox 中 **每个模块** 对应的 OpenAPI/Swagger 文件到 **GitHub 仓库**
* 支持自动备份 Apifox 中 **每个模块** 对应的 OpenAPI/Swagger 文件到 **GitLab 仓库**
* 备份会在**晚间的空闲时间**执行（具体时间不固定，为随机空闲时段）

## 新建 Git 仓库连接


<Steps>
  <Step>
    打开 **「项目设置」 → 「Git 仓库连接」**
      
<Background>

![CleanShot 2025-08-14 at 11.56.38@2x.png](https://api.apifox.com/api/v1/projects/5097254/resources/555700/image-preview)
</Background>

  </Step>
  <Step>
    点击 **「新建连接」**，跳转到网页后登录到对应的代码管理平台
      
  </Step>
  <Step>
    填写配置信息：

   * **GitHub**：名称、说明、GitHub 组织、GitHub 仓库
   * **GitLab**：名称、说明、GitLab 群组、GitLab 项目
      
   
<Background>
 
![image.png](https://api.apifox.com/api/v1/projects/5097254/resources/555703/image-preview)
</Background>

  </Step>
    <Step>
    保持仓库连接的配置信息
      
  </Step>
</Steps>


## 启用自动备份

<Steps>
  <Step>
   在需要备份的模块中，进入 **「概览」 → 「导出/备份」**，添加一个 OAS
      
<Background>

![CleanShot 2025-12-23 at 17.30.59@2x.png](https://api.apifox.com/api/v1/projects/5097254/resources/606683/image-preview)
</Background>

  </Step>
  <Step>
   启用 **「备份到 Git 仓库」**
  </Step>
  <Step>
    配置以下信息：

   * **Git 仓库连接**（选择已创建的连接）
   * **Git 分支**
   * **目标文件路径**（OpenAPI Spec 文件在仓库中的存放路径）
   
<Background>

![CleanShot 2025-08-14 at 12.02.21@2x.png](https://api.apifox.com/api/v1/projects/5097254/resources/555707/image-preview)
</Background>

  </Step>
</Steps>



保存后，系统将在晚间的随机空闲时间，自动将该模块的 **OpenAPI Spec 文件** 推送到指定 Git 仓库。

<Background>

![CleanShot 2025-08-14 at 12.04.40@2x.png](https://api.apifox.com/api/v1/projects/5097254/resources/555708/image-preview)
</Background>

## 导入 OpenAPI/Swagger 文件

支持从 GitHub、GitLab 仓库导入 OpenAPI/Swagger 文件，你可以在 “项目设置 -> 绑定数据源” 里选择 “Git 仓库” 作为数据源。
