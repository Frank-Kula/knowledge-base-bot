# 导入 Apizza



支持导入 Apizza 中的 `json` 文件，有关于 Apizza 产品的详情说明请参考[项目地址](https://wiki.apizza.net/projects/imexport)。

## 导出数据

访问 Apizza，在 “开发模式” 下，点击右侧的 “圆形花状” 按钮，轻点菜单下拉栏的 “导出” 选项。


<Background>

![image.png](https://api.apifox.com/api/v1/projects/5097254/resources/488851/image-preview)
</Background>


选择导出为 json 格式。

<Background>

![image.png](https://api.apifox.com/api/v1/projects/5097254/resources/488852/image-preview)
</Background>


## 导入数据

打开 Apifox 中的 “项目设置” 面板，点击 “导入数据”，选择 Apizza，上传已导出的 json 文件。
