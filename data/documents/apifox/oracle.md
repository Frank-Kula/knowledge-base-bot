# Oracle

## Windows 版

1. 访问官方[下载地址](https://www.oracle.com/database/technologies/instant-client/downloads.html)，选择对应 `Windows` 版本的 `Instant Client ZIP` 文件并下载。
2. 解压ZIP 文件到指定目录，如： `C:\oracle\instantclient_19_3`。
3. 将上述指定目录添加到环境变量 `PATH` 里（详情请查看[参考文档](https://jingyan.baidu.com/article/8ebacdf02d3c2949f65cd5d0.html)）。
4. 重启 Apifox。
