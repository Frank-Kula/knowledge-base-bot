# 下载 Apifox

相比 [Web 版](https://app.apifox.com/)，我们更推荐你使用 Apifox 客户端 *（桌面版）*。由于浏览器的安全限制，Web 版在调试时无法调用数据库、执行本地代码，而且 GET 和 HEAD 方法的请求不支持携带请求 Body。使用 Apifox 客户端，你可以：

- 发送请求
- 设计 API
- 调试 API
- 测试 API
- 使用 Mock 数据
- 发布 API 文档

你可以根据操作系统选择合适的安装包，并在[官网首页](https://apifox.com/)或下方表格中点击下载。


<Background>
  ![Apifox 下载](https://api.apifox.com/api/v1/projects/5097254/resources/465385/image-preview)
</Background>


Apifox 兼容 32 位和 64 位 Windows 系统，支持 Intel 和 Apple 芯片的 macOS，还适用于 Linux 系统。

## 常见问题


<Accordion title="如何下载 Alpha 版？" defaultOpen>
1. Apifox 新功能都会先在 Alpha 版上线，等稳定后才会合到正式版。

2. Alpha 版可能会有一些 bug，如遇到问题请在[交流群](https://docs.apifox.com/contact-us.md)里反馈，我们会第一时间解决。

3. Alpha 版和正式版数据是互通的。

4. 已经是 Alpha 版的，直接点击软件内更新。

5. Alpha 版下载地址：
https://docs.apifox.com/download-alpha
</Accordion>


<Accordion title="为什么下载时会跳到百度网盘或天翼网盘？" defaultOpen={false}>
1. 你所在的 IP 地址下载 Apifox 软件的次数过于频繁，触发了安全限制，需切换 IP 方可直接下载安装包。
2. 如果你的电脑开启了代理，下载时会通过百度网盘或天翼网盘的通道进行，而不是直接下载安装包。如果觉得麻烦，可以先关闭代理。
</Accordion>
