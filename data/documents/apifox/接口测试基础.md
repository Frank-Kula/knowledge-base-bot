# 接口测试基础

**来源**: https://docs.apifox.com/help

## 简介

Apifox 是一个 API 开发协作平台，支持接口设计、调试、文档生成和测试。

## 发送 HTTP 请求

### 基本步骤

1. 创建新项目
2. 添加接口
3. 设置请求方法和 URL
4. 添加请求参数
5. 发送请求

### GET 请求

GET 请求用于获取数据。示例：

```
GET https://api.example.com/users
```

### POST 请求

POST 请求用于创建数据。示例：

```
POST https://api.example.com/users
Content-Type: application/json

{
  "name": "张三",
  "email": "zhangsan@example.com"
}
```

## 常见问题

### 接口测试失败怎么办？

1. 检查网络连接
2. 确认服务器地址正确
3. 查看错误码和错误信息
4. 检查请求参数是否正确

### 返回 500 错误

500 错误通常是服务器内部错误，可能的原因：
- 服务器代码异常
- 数据库连接失败
- 服务器资源不足

建议：
- 检查服务器日志
- 联系后端开发人员
- 确认是否有相关 Bug 报告
