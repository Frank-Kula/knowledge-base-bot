# 快捷分享 API 版本



在快捷分享中，也可以新建针对于某个 API 版本中的接口进行单次在线分享。


<Background>

![CleanShot 2024-12-19 at 16.40.38@2x.png](https://api.apifox.com/api/v1/projects/5097254/resources/484974/image-preview)
</Background>


在新建分享时，可以选择本次快捷分享的分享接口范围。接口范围的选择可以选中具体某一个 API 版本，然后从此 API 版本中选择全部或部分接口生成在线分享链接。


<Background>

![image.png](https://api.apifox.com/api/v1/projects/5097254/resources/484977/image-preview)
</Background>


外部用户在访问这个在线分享链接时，即可看到指定 API 版本中的全部或部分接口内容。

<Background>

![CleanShot 2024-12-19 at 16.43.26@2x.png](https://api.apifox.com/api/v1/projects/5097254/resources/484979/image-preview)
</Background>
