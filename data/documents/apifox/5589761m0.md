# 人物（Person）

生成人物个人信息的模块，例如姓名和职位。

<br />

**概述**

要生成全名，请使用 `{{$person.fullName}}`。请注意，这与简单地连接 `{{$person.firstName}}` 和 `{{$person.lastName}}` 不同，因为全名可能包含前缀、后缀或两者。此外，不同的支持语言环境将具有不同的命名模式。例如，姓氏可能出现在名字之前，或者名字或姓氏可能是双重或连字符的。

<br />

你也可以使用 `{{$person.prefix}}`、`{{$person.firstName}}`、`{{$person.middleName}}`、`{{$person.lastName}}` 和 `{{$person.suffix}}` 分别生成姓名的各个部分。并非所有语言环境都支持所有这些部分。

<br />

此模块中的许多方法可以选择女性、男性或混合姓名。

<br />

还提供与工作相关的数据。要生成职位，请使用 `{{$person.jobTitle}}`。

<br />

此模块还可以生成其他可能出现在用户资料中的个人信息，例如 `{{$person.gender}}`、`{{$person.zodiacSign}}` 和 `{{$person.bio}}`。

<br />

**相关模块**

有关个人联系信息（如电话号码和电子邮件地址），请参阅[电话（Phone）](https://docs.apifox.com/5589763m0.md)和[网络（Internet）](https://docs.apifox.com/5589698m0.md)模块。



## bio

返回一个随机的简短个人简介

**返回**: 字符串

**示例**

```js
{{$person.bio}}  // '涂鸦爱好者，歌手 🥱'
```


## firstName

返回一个随机的名字。

**参数**

| 名称 | 类型               | 默认值 | 描述                                                |
| ---- | ------------------ | ------ | --------------------------------------------------- |
| sex  | 'female' \| 'male' |        | 可选的要使用的性别。可以是 `'female'` 或 `'male'`。 |

**返回**: 字符串

**示例**

```js
{{$person.firstName}}  // 'Pierce'

{{$person.firstName(sex='female')}}  // 'Elisa'

{{$person.firstName(sex='male')}}  // 'Saul'
```


## fullName

生成一个随机的全名。

**参数**

| 名称      | 类型               | 默认值                                         | 描述                                                |
| --------- | ------------------ | ---------------------------------------------- | --------------------------------------------------- |
| firstName | 字符串             | `{{$person.firstName}}`                        | 可选的要使用的名字。如果未指定，将随机选择一个。    |
| lastName  | 字符串             | `{{$person.lastName}}`                         | 可选的要使用的姓氏。如果未指定，将随机选择一个。    |
| sex       | 'female' \| 'male' | `{{$helpers.arrayElement(['female','male'])}}` | 可选的要使用的性别。可以是 `'female'` 或 `'male'`。 |

**返回**: 字符串

**示例**

```js
{{$person.fullName}}  // 'Jerald Hirthe'

{{$person.fullName(firstName='Joann')}}  // 'Joann Buckridge'

{{$person.fullName(firstName='Marcella',sex='female')}}  // 'Marcella O'Keefe-Trantow'
```

## sex

返回一个随机的性别。

此方法的输出已本地化，因此不应使用它来填充例如 `{{$person.firstName}}` 等其他模块中可用的参数 `sex`。

**返回**: 字符串

**示例**

```js
{{$person.sex}} // '女性'

{{$person.prefix(sex='female')}} // '夫人'
```
