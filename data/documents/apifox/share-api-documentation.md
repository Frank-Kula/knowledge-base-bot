# 分享 API 文档

当 API 开发完成并调试成功后，你可以非常方便地将它发布为 API 文档。

<Steps>
  <Step>
切换到 **“分享文档”** 模块。
      
<Background>

![CleanShot 2025-11-05 at 15.25.16@2x.png](https://api.apifox.com/api/v1/projects/5097254/resources/588474/image-preview)
</Background>

    </Step>
  <Step>
点击 **“新建分享”** 以创建一个新的分享文档。
    </Step>
  <Step>
给它起个标题，选择所有环境，然后点击 **“保存”**。
<Background>
   
    <img src="https://api.apifox.com/api/v1/projects/5097254/resources/465706/image-preview" style="width:500px" />
</Background>
    </Step>
  <Step>
现在你可以在列表中看到这个分享项，点击 “打开链接” 以在网页中打开。
    </Step>
  <Step>
你可以在左侧的目录树中切换不同的接口。对于每个接口，你可以点击 **“调试”** 按钮发送请求，还可以生成示例代码。
<Background>
    
![CleanShot 2024-09-14 at 15.03.50@2x.png](https://api.apifox.com/api/v1/projects/5097254/resources/465707/image-preview)

</Background>
    </Step>
</Steps>

:::tip[]
Apifox 还支持自定义文档布局、自定义域名、自定义 URL 等更多功能。了解更多关于[发布文档站](https://docs.apifox.com/publish-documentation-site.md)的信息。
:::
