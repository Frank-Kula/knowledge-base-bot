# 数组如何配置枚举？

### 场景一：字段是 **字符串数组**，并限制只能是固定值中的某几个

比如，一个字段是数组类型，每个元素只能是 `"A"`、`"B"` 或 `"C"`：

```json
{
  "type": "array",
  "items": {
    "type": "string",
    "enum": ["A", "B", "C"]
  }
}
```

这种写法表示：**数组本身可以有多个元素，但每个元素必须是 `"A"`、`"B"`、`"C"` 之一。**

你可以在高级设置的 JSON Schema 编辑器中输入枚举：

### 场景二：限制整个数组的取值是固定的几种（枚举整个数组的取值）

如果你想枚举整个数组的组合（比如只能是 `[1, 2]` 或 `[3, 4]`），可以这样：

```json
{
  "type": "array",
  "enum": [
    [1, 2],
    [3, 4]
  ]
}
```

这种写法表示：**整个数组的值只能是 `[1, 2]` 或 `[3, 4]` 之一**。


这种场景只支持在高级设置的 JSON Schema 编辑器中输入枚举。
