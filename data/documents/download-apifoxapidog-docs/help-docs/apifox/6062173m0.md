# Apifox 中如何设置请求超时时间？


<Steps>
  <Step>
    点击 Apifox 页面右上角的 “设置” icon。
      
<Background>

![CleanShot 2025-02-24 at 12.21.10@2x.png](https://api.apifox.com/api/v1/projects/5097254/resources/497689/image-preview)
</Background>

  </Step>
  <Step>
    在弹出的 “设置” 界面，点击 “通用” 选项，你会在右边的选项卡中看到 “请求超时时间” 的设置，填写即可。超时时间默认为 300000 毫秒（5 分钟），如果设置为 0，则不限制超时时间。
      
<Background>

![CleanShot 2025-02-24 at 12.22.54@2x.png](https://api.apifox.com/api/v1/projects/5097254/resources/497690/image-preview)
</Background>

  </Step>
</Steps>
