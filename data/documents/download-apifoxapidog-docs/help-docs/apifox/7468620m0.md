# 解决 ENOTFOUND: getaddrinfo ENOTFOUND www 错误

典型的 **域名解析（DNS）错误**，和 [`ENOTFOUND: Couldn't resolve host`](https://docs.apifox.com/6534745m0) 是同一类问题。

`getaddrinfo ENOTFOUND www` 意味着系统在尝试解析你请求的域名时，**找不到名为 "www" 的主机**。


## 错误原因

这个错误几乎可以确定是 **URL 格式不完整或错误** 导致的。

1.  **URL 不完整（最可能的原因）：** 你在 Apifox 中发送请求时，**只填写了 `www`**，而没有写完整的域名，比如你本来想请求的是 `http://www.baidu.com`，但只填了 `www`。
2.  **缺少协议头：** 即使你写了完整的域名，但可能缺少了 **`http://`** 或 **`https://`** 协议头，导致系统无法识别这是一个完整的网络地址。


## 解决方法

你只需要专注于修正你在 Apifox 中的请求地址即可。

在 Apifox 的请求地址栏中，确保你填写的是一个完整的、可解析的地址。确保地址中包含 **`http://`** 或 **`https://`** 协议头。
