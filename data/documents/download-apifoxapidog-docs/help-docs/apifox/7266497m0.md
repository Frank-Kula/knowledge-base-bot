# 如何在 Apifox 中的同一字段处理字符串和文件类型？

Apifox 支持在同一字段中处理字符串和文件类型。您可以通过字段-数据类型右侧的“高级设置”按钮，选择 JSON Schema，然后粘贴以下内容来实现：

```
{
    "oneOf": [
        {
            "type": "string",
            "format": "uri"
        },
        {
            "type": "string",
            "format": "binary"
        }
    ]
}
```
