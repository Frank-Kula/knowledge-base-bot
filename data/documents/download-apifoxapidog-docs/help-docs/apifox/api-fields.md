# 接口字段

在 API 协作中，常常需要添加自定义字段来补充接口描述。例如，可以添加 “是否需要授权”、“响应格式” 或 “速率限制” 等字段，以便为 API 使用者提供更多背景信息。

在 Apifox 中，你可以使用自定义字段功能，轻松添加这些附加信息。

## 配置自定义字段

你可以导航到 `项目设置` > `通用设置` > `功能设置` > `接口功能设置` 来设置 Apifox 中的接口字段。你在这里配置的自定义字段将在整个项目中生效。


<Background>

![image.png](https://api.apifox.com/api/v1/projects/5097254/resources/477511/image-preview)
</Background>


Apifox 提供了三个默认的内置字段，你可以手动启用或禁用这些字段。如果启用，它们将出现在接口元数据中。


<Background>

![image.png](https://api.apifox.com/api/v1/projects/5097254/resources/477512/image-preview)
</Background>


### 添加新字段

你还可以根据需要灵活地添加自定义字段。


<Background>
<p style="text-align:center">
    <img src="https://api.apifox.com/api/v1/projects/5097254/resources/477513/image-preview" style="width:340px" />
</p>
</Background>



在 Apifox 中，字段类型支持以下几种：`文本`、`数字`、`单选`、`多选`、`日期`、`项目成员`、`链接`、`邮箱`、`单标签` 和 `多标签`。


### 在 API 文档中显示字段

在 “分享文档 -> 快捷分享”中，你可以选择是否显示这些自定义字段。

<Steps>
  <Step>
    在 Apifox 创建快捷分享时，你可以选择 “展示项目字段” 选项，以包含你定义的接口字段。
     
<Background>
 <p style="text-align:center">
    <img src="https://api.apifox.com/api/v1/projects/5097254/resources/477514/image-preview" style="width:440px" />
</p>
</Background>


  </Step>
  <Step>
    选择要显示的字段。
      
<Background>
<p style="text-align:center">
    <img src="https://api.apifox.com/api/v1/projects/5097254/resources/477515/image-preview" style="width:540px" />
</p>  
</Background>

   
  </Step>
  <Step>
    它们将显示在文档中。
  </Step>
</Steps>

在 Apifox 发布文档时，你也可以选择要显示哪些字段。在 `分享文档` 部分，导航到 `发布文档` > `个性化设置` > `外观&展示`，你可以选择字段。
