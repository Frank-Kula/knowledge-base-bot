# 音乐（Music）

生成音乐相关条目的模块。

<br />

**概述**

生成随机音乐内容。

要获取随机专辑名称，请使用 `{{$music.album}}`。

要获取随机艺术家，请使用 `{{$music.artist}}`。

要获取随机音乐类型，请使用 `{{$music.genre}}`。

要获取随机歌曲名称，请使用 `{{$music.songName}}`。

所有数据类型都可以本地化。



## album

返回随机专辑名称。

**返回**: 字符串

**示例**

```js
{{$music.album}}  // 'El Viaje De Copperpot'
```


## artist

返回随机艺术家姓名。

**返回**: 字符串

**示例**

```js
{{$music.artist}}  // 'Chayanne'
```


## genre

返回随机音乐类型。

**返回**: 字符串

**示例**

```js
{{$music.genre}}  // '酸爵士'
```


## songName

返回随机歌曲名称。

**返回**: 字符串

**示例**

```js
{{$music.songName}}  // 'Bad Romance'
```
