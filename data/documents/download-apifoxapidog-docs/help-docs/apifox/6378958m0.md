# 如何在 PowerShell 中运行 Apifox CLI 命令

PowerShell 运行 Apifox CLI 命令 报错`apifox: could not find "html cli" reporter`


![image.png](https://api.apifox.com/api/v1/projects/5097254/resources/509372/image-preview)
PowerShell 不支持 html,cli 这个传参，可以尝试加个引号`'html,cli'`再传参，例如：

```js
apifox run --access-token APS-xxx -t xx -e xx -n 1 -r 'html,cli'
```
