# 新建迭代分支


当进入一个新的迭代，迭代需求已经明确，需要进入技术设计流程时，进入 Apifox 中的对应项目，在目录树顶端右上角的迭代分支切换功能中，即可点击“创建迭代分支”功能，建立一个新的迭代分支开启迭代相关工作。


<Background>

![image.png](https://api.apifox.com/api/v1/projects/5097254/resources/484986/image-preview)
</Background>


当然也可点击管理迭代分支功能查看迭代分支列表，或者进入「项目设置 -> 管理迭代分支」功能中，查看更多迭代分支的细节，并且选择创建一个新的迭代分支。
