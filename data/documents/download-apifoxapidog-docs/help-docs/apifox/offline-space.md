# 离线空间

Apifox 的离线空间是一个全新的本地化 API 调试功能，目前正在内测阶段。该功能支持在完全离线的环境下进行 API 调试，所有数据均存储在本地，确保数据安全性和私密性。

## 功能特点

- **离线调试**：无需网络连接即可进行 API 调试
- **本地存储**：所有数据保存在本地，保障数据安全
- **简单快捷**：操作简便，快速上手
