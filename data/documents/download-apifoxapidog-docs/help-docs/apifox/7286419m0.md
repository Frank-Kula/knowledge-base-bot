# 为什么点击 Body 的自动生成 点击不生效，不更新参数值？

显示如下图所示：


<Background>
![img_v3_02pg_7ffff954-c8a9-4c41-81b5-9be593eb526g.png](https://api.apifox.com/api/v1/projects/5097254/resources/566270/image-preview)
</Background>



在 Body 区域点击「自动生成设置」，确认是否需要切换到其他配置。

常见原因包括：
1. 只设置了 body 参数，但未填写示例值；
2. 只添加了参数，未配置示例；
3. 勾选了「仅生成字段名」，因此不会自动生成对应的值。
