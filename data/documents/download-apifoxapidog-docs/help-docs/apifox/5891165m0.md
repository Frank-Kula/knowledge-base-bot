# 为什么 Markdown 文档中使用多个 $ 时部分内容没有正确显示？

Apifox 使用的 Markdown 编辑器会将`$xxxxx$`解析为数学公式。如果你的内容并非数学公式，但是包含了多个 `$` 可能导致解析失败，进而无法正确显示。
为了避免这种情况，可以采用以下方法：
#### 方法一. 使用行内代码块
将代码内容用反引号 ` 包括起来，例如

</Columns>

</Container>

#### 方法二. 使用转义字符
