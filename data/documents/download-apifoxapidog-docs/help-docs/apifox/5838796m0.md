# 怎么在 Apifox 的 Markdown 中使用透明背景图片？

因为大多数透明背景的 PNG 图片在暗黑模式下显示效果不好，所以 Apifox 的 Markdown 默认会在暗黑模式下给透明 PNG 添加白色背景。

比如：


![image.png](https://api.apifox.com/api/v1/projects/5097254/resources/490400/image-preview)


如果你想显示透明背景的 PNG，可以在 Markdown 中使用下面的语法：

```html
<img src="https://api.apifox.com/api/v1/projects/5097254/resources/490403/image-preview" style="background-color: transparent; width: 64px"/>
```

这段 HTML 代码在 Apifox 的 Markdown 中会显示成这样：

<img src="https://api.apifox.com/api/v1/projects/5097254/resources/490403/image-preview" style="background-color: transparent; width: 64px"/>

这种方法让你可以在 Apifox 的 Markdown 中显示透明背景的 PNG 图片，而不会在暗黑模式下应用默认的白色背景。透明背景会被保留，在明亮模式和暗黑模式下都能保持一致的外观。

这样可以确保图片显示的是透明背景，不管用户是在明亮模式还是暗黑模式下。`style="background-color: transparent;"`属性覆盖了 Apifox Markdown 的默认行为，允许图片以原本的透明背景显示。

你还可以通过添加`width: 64px`样式属性来调整图片的宽度，或者设置其他你想要的宽度。
