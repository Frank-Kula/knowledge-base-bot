# 如何使用数据库查询得到的数据循环传参给接口运行?

Apifox 支持使用数据库查询结果循环传参给接口运行。您可以按照以下步骤操作：

<Steps>
  <Step title="添加数据库操作步骤">
    在测试场景中添加一个“数据库操作”步骤。执行 SQL 查询，并将查询结果提取到一个变量中（例如：`id` ）。确保 SQL 查询返回的是一个数组格式的结果。
<Background>
  ![](https://api.apifox.com/api/v1/projects/5097254/resources/492681/image-preview)
</Background>
  </Step>
  <Step title="添加 Foreach 循环器">
    在测试场景中添加一个“Foreach 循环”步骤。将“待循环的数组”设置为之前提取的变量，例如 `{{id}}`。 循环的当前元素会自动存储在 `$item` 变量中。
<Background>
  ![](https://api.apifox.com/api/v1/projects/5097254/resources/492682/image-preview)
</Background>
  </Step>
  <Step title="在 Foreach 循环步骤下添加需要循环运行的接口">
    在接口请求的参数中，使用`{{$.步骤id.element}}`的方式引用循环变量的值。例如，如果数据库查询结果 id 是一个包含 userId 字段的对象数组，Foreach 循环步骤的步骤 id 为 2 ，则可以使用 `{{$.2.element}}` 作为接口参数的值。
<Background>
  ![](https://api.apifox.com/api/v1/projects/5097254/resources/492683/image-preview)
</Background>
  </Step>
</Steps>
