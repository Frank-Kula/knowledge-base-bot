# 分享文档

在 Apifox 中设计或开发完 API 后，你可以方便快捷地把 API 分享给其他同事，或者将其发布到公网。



## 文档发布方式

Apifox 提供了两种发布文档的方式。

1. **快捷分享**

一般用于团队内部或跨团队协作，可以把部分 API 文档分享给协作者。

2. **发布文档站**

可以发布多个在线文档站，支持设置访问权限。

功能 | 快捷分享 | 发布文档
--- | --- | ---
发布部分 API | 支持 | 支持，从第二个文档站开始，可以自动圈选要分享的接口
自定义域名 | 不支持 | 支持
自定义导航 | 不支持 | 支持
自定义颜色和 Logo | 不支持 | 支持
发布到 API Hub | 不支持 | 支持


:::info[]
Apifox 的快捷分享与发布文档站功能均设有 WAF（Web 应用程序防火墙）防护层。

该防护层包含 CC（Collapse Challenge）防护策略，用于抵御恶意访问。当系统监测到特定 IP 地址存在访问可疑 URL 或在短时间内发起高频请求等行为时，可能会触发风险控制机制。一旦触发，该 IP 地址将在接下来的一段时间内无法访问 Apifox 的快捷分享页面或文档站点。
:::
