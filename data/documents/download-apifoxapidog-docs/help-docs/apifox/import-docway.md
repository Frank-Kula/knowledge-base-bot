# 导入小幺鸡（docway）


支持识别 DOCWAY 的文档分享地址并导入至 Apifox。

## 导出数据

打开 DOCWAY 进入对应项目，将鼠标移到「更多功能」并点击「分享项目」（仅“项目创建者”有权限分享项目）。


<Background>
 <img src="https://api.apifox.com/api/v1/projects/5097254/resources/488845/image-preview" width="360px" />
</Background>


创建分享地址并将“阅读密码”设置为空，然后“复制分享地址”。

## 导入数据

打开 Apifox 中的 “项目设置” 面板，点击 “导入数据”，选择小幺鸡（docway）并以 URL 导入，将复制的分享地址黏贴到输入框中，点击 “导入” 即可。
