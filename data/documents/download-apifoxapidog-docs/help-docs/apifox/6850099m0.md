# 内网私有部署的大模型，能够直接用 apifox 的 mcp 嘛？

**Q：电脑是否能访问外网？**

首先，能不能用 MCP，取决于你用的 AI 客户端支不支持 MCP。

其次，如果用的是 SaaS 版 Apifox，MCP 需要访问 Apifox 服务器获取数据。所以才问能否访问外网。

**Q：如果我的客户端支持 mcp，且是纯内网环境的话，apifox 是否能支持本地部署。**

本地部署需要购买。如果是私有化部署版本，那么 MCP 就会请求本地部署的服务器了。
