# 食物（Food）

用于生成与食物相关数据的模块。

<br />

**概述**

此模块提供生成各种与食物相关信息的方法，例如菜单上的项目。要生成菜肴的名称，请使用 `dish()`，要生成菜肴的长描述，请使用 `description()`。请注意，这些不会相互对应。你还可以生成菜肴的单个成分，例如香料、蔬菜、肉类、水果或通用成分。



## adjective

生成一个随机的菜肴形容词。

**返回**: 字符串

**示例**

```js
{{$food.adjective}}  // '酥脆的'
```


## description

生成一个随机的菜肴描述。

**返回**: 字符串

**示例**

```js
{{$food.description}} // '用浓郁的咖喱和 rhubarb 酱汁精心炖制的鸭肉，配以奶油鲜辣椒。'
```


## dish

生成一个随机的菜肴名称。

**返回**: 字符串

**示例**

```js
{{$food.dish}} // '三文鱼握寿司'
```


## ethnicCategory

生成随机食物的民族类别。

**返回**: 字符串

**示例**

```js
{{$food.ethnicCategory}} // '波斯'
```


## fruit

生成一个随机的水果名称。

**返回**: 字符串

**示例**

```js
{{$food.fruit}} // '蜜瓜'
```


## ingredient

生成一个随机的成分名称。

**返回**: 字符串

**示例**

```js
{{$food.ingredient}} // 'garam masala'
```


## meat

生成一个随机的肉类

**返回**: 字符串

**示例**

```js
{{$food.meat}} // '鸡肉'
```


## spice

生成一个随机的香料名称。

**返回**: 字符串

**示例**

```js
{{$food.spice}} // '豆蔻'
```


## vegetable

生成一个随机的蔬菜名称。

**返回**: 字符串

**示例**

```js
{{$food.vegetable}} // '菊苣'
```
