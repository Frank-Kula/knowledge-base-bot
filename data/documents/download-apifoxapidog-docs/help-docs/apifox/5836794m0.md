# Web 端与客户端有何区别？

Web 端与客户端在主要流程的使用上没有明显差异，都能够满足团队内的接口协作需求，但以下功能存在差异。

| 功能               | Web 端支持 | 客户端支持 |
|--------------------|------------|------------|
| **导出接口**       | ❌         | ✅         |
| **本地 Mock 功能** | ❌         | ✅         |
| **生成业务代码**   | ❌         | ✅         |
| **外部程序**       | ❌         | ✅         |
| **调整字体大小**   | ❌         | ✅         |
| **网络代理**       | ❌         | ✅         |
