# 总览

Apifox 已于 `2.7.18` 版本在客户端内开始接入 AI 能力，期望通过 AI 来帮助用户更高效的进行 API 的内容生成、测试和发布。

你可以使用各个 AI 模型供应商提供的 API Key 来接入 AI（需先 [开启 AI 功能](https://docs.apifox.com/enable-ai-features.md)）。

- AI 功能依赖你自行配置的 AI 模型提供商，你可以选择任意 AI 模型提供商，并使用自己的 API key。
- Apifox 不提供 AI 模型，也不会使用你的数据进行训练。

**已经上线并可使用的 AI 功能有：**

- 辅助修改数据模型，增加描述、Mock 等信息；
- 检测 API 文档的内容规范性；
- AI 辅助命名 API 接口字段；
- AI 生成测试用例，支持同步生成测试数据、前后置操作（断言和自定义脚本）；
- 检测 API 文档完整性；


**计划中的 AI 功能有：**

- 自动生成接口用例；

- ...

我们会逐步完善、扩展 AI 在 Apifox 中的使用能力，并提高相关能力的使用体验。如果你有更好的建议，欢迎在交流群给我们进行反馈！
