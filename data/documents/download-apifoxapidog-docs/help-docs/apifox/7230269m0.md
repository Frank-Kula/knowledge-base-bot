# 接口返回的是 Protobuf 数据，如何正常查看？

可以通过后置脚本结合`protobufjs`库解析，将二进制`Protobuf`数据转换为`JSON`格式，步骤如下：
1. 使用后置脚本加载`protobufjs`库
2. 引入对应的`.proto`文件定义
3. 解析接口返回的二进制流数据

示例脚本：
```js
fox.liveRequire("protobufjs", (protobuf) => {
  try {
    // 加载proto文件
    const proto = `
    syntax = "proto3";
    message TestMessage {
        float number  = 1;
        repeated string texts = 2;
    }
    `
    // 获取原始响应
    const stream = pm.response.stream;
    console.log(stream)
    // 反序列化
    const root = protobuf.parse(proto).root
    const TestMessage = root.lookupType("TestMessage")
    const json = TestMessage.decode([...stream])
    // 修改响应
    pm.response.setBody(json);
  } catch (error) {
    console.error("An error occurred during liveRequire callback", error.stack)
    throw error
  }
});
```
