# 如何邀请已加入团队的成员进入项目？

1. 项目管理员进入成员管理页
打开项目，进入`项目设置`-`成员管理`页面，点击右上角`添加`
<Background>
![image.png](https://api.apifox.com/api/v1/projects/5097254/resources/509244/image-preview)
</Background>
2. 在弹出的窗口中可以查看已经加入团队的所有成员，选择你想邀请的成员（支持多选），点击`邀请`。该成员即成功加入团队中
<Background>
![image.png](https://api.apifox.com/api/v1/projects/5097254/resources/509246/image-preview)
</Background>


:::caution[]
注意：上述操作仅适用于已加入团队的成员。如果成员尚未加入团队，可以通过项目侧边栏的`邀请成员`进行邀请。

![image.png](https://api.apifox.com/api/v1/projects/5097254/resources/509248/image-preview)
:::
