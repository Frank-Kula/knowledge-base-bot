# 历史记录

`历史记录`记录了团队成员对 API 各个字段所做的修改，以及由于 `导入覆盖` 产生的调整。

它还提供了一个 `对比` 工具，可以在修改前后对版本进行对比，突出显示差异，并支持恢复到任意一个历史版本。

要访问 `历史记录`，点击 `API` 区域右上角的图标，展开历史记录视图。


<Background>

![image.png](https://api.apifox.com/api/v1/projects/5097254/resources/476116/image-preview)
</Background>


你可以查看接口的修订历史，包括贡献者和时间戳信息。

<Background>
<p style="text-align: center">
    <img src="https://api.apifox.com/api/v1/projects/5097254/resources/476119/image-preview" style="width: 340px" />
</p>
</Background>

选择一条记录后，会显示修改前后的各个字段变化。


<Background>

![image.png](https://api.apifox.com/api/v1/projects/5097254/resources/476120/image-preview)
</Background>


要恢复到之前的版本，只需点击 `还原此版本`。此操作会基于所选历史版本创建一个新版本，同时保留原始的修改历史。
