# 团队变量



> [Apifox 商业专业版及以上版本可使用团队变量功能](https://apifox.com/pricing)


可以在 “Apifox 主窗口 -> 团队资源 -> 变量” 中添加团队变量，用来跨项目传递数据。适用于项目与项目之间的变量流转需求，例如微服务场景。你可以在每个项目的 [“环境管理 -> 全局变量 -> 团队中共享的全局变量”](https://docs.apifox.com/global-environment-session-variables.md) 面板里查看到已设置的团队变量，并在项目内使用团队变量。

**注意：**

1. 在此处可以任意添加、修改、删除团队变量，点击保存按钮后页面内容才会正式保存。
2. 这里仅可设置团队变量的 “远程值”，用来给实际在某个项目中使用此团队变量时有一个默认值。
3. 在团队内的项目中，是无法通过手动或者脚本的方式增加、更改、删除团队变量的，只能更改指定团队变量的`本地值`。通过这样来防止因为团队变量的范围过大，导致大多数团队成员可以任意更改团队变量而不好管理。
