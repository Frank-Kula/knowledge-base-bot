# 数据类型（Datatype）

生成布尔值的模块。

<br />

**概述**

对于简单的随机真或假值，请使用 `boolean()`。



## boolean

返回布尔值 true 或 false。

注意：概率为 `0.75` 会导致 75% 的调用返回 `true`；同样，`0.3` => `30%`。如果概率 `<= 0.0`，它将始终返回 `false`。如果概率 `>= 1.0`，它将始终返回 `true`。概率限制为两位小数。

**参数**

| 名称        | 类型 | 默认值 | 说明                                  |
| ----------- | ---- | ------ | ------------------------------------- |
| probability | 数字 | `0.5`  | 返回 `true` 的概率 (`[0.00, 1.00]`)。 |

**返回**: 布尔值

**示例**

```js
{{$datatype.boolean}}  // 'false'

{{$datatype.boolean(probability=0.1)}}  // ‘false’
```

:::tip[]
函数 `faker.datatype.boolean(0.9)` 使用位置参数 `0.9` 来指定返回 `True` 的概率。

这已更新为使用命名参数 `probability`。等效的函数调用现在是 `{{ $datatype.boolean(probability=0.9) }}`。
:::
