# 如何检查 CNAME 配置是否正确？

- Linux 使用 `dig` 命令检查域名解析是否正常。  
```shell
dig [你的自定义域名]
```
    
- Windows 系统需要安装 `dig` 命令，参考此文档安装：[安装 dig 命令](https://bbs.sangfor.com.cn/forum.php?mod=viewthread&tid=139699)  

命令运行后，查看 `ANSWER SECTION` 部分是否与 Apifox 上的信息配置一致，例如：
