# 如何控制 Markdown 表格的列宽？

你可以使用 HTML 标签中的 `div` 元素，通过行内样式来调整表格列的宽度。例如：

    ```
    | <div style="width: 90px;">姓名</div> | 年龄 | 性别 | 邮箱 |  
    | --- | --- | --- | --- |  
    | 张三 | 25 | 男 | [zhangsan@example.com](mailto:zhangsan@example.com) |  
    | 李四 | 30 | 女 | [lisi@example.com](mailto:lisi@example.com) |  
    | 王五 | 35 | 男 | [wangwu@example.com](mailto:wangwu@example.com) |
    ```
