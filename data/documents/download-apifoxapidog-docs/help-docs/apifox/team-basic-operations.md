# 团队基本操作


在 Apifox 中，管理团队模块是一个非常重要的功能，它可以帮助你轻松地管理你的团队成员，分配任务和权限，以及查看工作的进度。下面将为你介绍如何使用 Apifox 的管理团队模块。

## 创建团队

在 Apifox 中，你可以创建多个团队，每个团队可以包含多个成员。要创建一个新团队，可以按照以下步骤操作：

1. 点击左侧导航栏中的“我的团队”选项。

2. 点击“新建团队”按钮。


<Background>
 <img src="https://api.apifox.com/api/v1/projects/5097254/resources/485319/image-preview" width="360px"/>
</Background>

3. 输入团队名称。


<Background>
 <img src="https://api.apifox.com/api/v1/projects/5097254/resources/485323/image-preview" width="360px"/>
    
</Background>


4.点击“新建”按钮。

现在，你已经成功创建了一个新团队，在页面左侧点击团队名称可进入不同的团队。

## 更改团队信息

团队所有者（或管理员）可通过 “团队设置 -> 基础信息”，修改团队名称。

更改团队信息后，所有团队成员都将看到新的团队信息。


<Background>

![image.png](https://api.apifox.com/api/v1/projects/5097254/resources/485329/image-preview)
</Background>

## 退出团队

团队成员可通过 “团队设置 -> 危险区域”，退出团队。


<Background>

![image.png](https://api.apifox.com/api/v1/projects/5097254/resources/485331/image-preview)
</Background>

:::tip[]
退出团队后，将失去该团队内所有项目的访问权限。
:::


## 移交团队

团队所有者可通过 “团队设置 -> 危险区域”，移交给团队其他成员。


<Background>

![image.png](https://api.apifox.com/api/v1/projects/5097254/resources/485332/image-preview)
</Background>


## 解散团队

团队所有者可通过 “团队设置 -> 危险区域 -> 解散团队”，将团队完全解散。

:::tip[]
解散团队后，该团队下的所有项目都将被同步删除，且不可恢复。
:::
