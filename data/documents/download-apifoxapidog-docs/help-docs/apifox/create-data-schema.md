# 新建数据模型

## 新建数据模型

<Steps>
  <Step>
    在 `接口管理` 模块中，点击`➕`图标并选择 `新建数据模型` 来创建一个新的数据模型。

<Background>

![CleanShot 2025-11-05 at 16.44.34@2x.png](https://api.apifox.com/api/v1/projects/5097254/resources/588577/image-preview)
</Background>

  </Step>
  <Step>
    填写数据模型的名称和描述。
  </Step>
  <Step>
    在数据模型编辑界面中[构建数据模型](https://docs.apifox.com/build-data-schemas.md)的结构。
  </Step>
</Steps>

## 管理数据模型

在 `接口管理` 模块的 `数据模型` 部分，你可以管理项目中的所有数据模型——新建数据模型、重命名、删除和重新排序。

你可以通过创建目录来组织数据模型，以便更好地进行结构化和管理。
