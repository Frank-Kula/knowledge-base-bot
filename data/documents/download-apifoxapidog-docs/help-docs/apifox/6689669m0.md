# 如何为 Apifox 中的团队配置「IP 允许访问名单」？

Apifox 支持为团队配置单个 IP 地址或 IP 范围（CIDR 表示法），限制只有在该名单上的 IP 才能访问团队资源。适合管理企业内网或特定办公网络的访问权限，为团队资产提供更精细的安全保护。
