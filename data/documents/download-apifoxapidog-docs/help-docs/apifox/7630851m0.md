# Apifox是否可以在打开新接口时，直接打开新的Tab，而不是替换当前Tab

这个问题我们需要先了解 Tab 的状态

1. Tab 内文本显示斜体为非固定的 Tab，会被新开资源替换
<Background>
![image.png](https://api.apifox.com/api/v1/projects/5097254/resources/589684/image-preview)
</Background>

2. Tab内文本显示正体为固定的 Tab，不会被新打开资源替换
<Background>
![image.png](https://api.apifox.com/api/v1/projects/5097254/resources/589683/image-preview)
</Background>


如何保持 Tab 的固定？
1. 在接口树时，直接双击打开资源
2. 非固定态的 Tab（显示斜体），双击 Tab 切换到固定态
