# 响应与 Cookie

API 响应是 API 服务器在处理客户端请求后返回的数据。Apifox 提供强大的功能，帮助你校验响应数据、创建可视化效果并排查问题。你还可以将响应保存为返回示例，以便将来参考，同时管理相关的 Cookie。

## 查看和保存响应

发送 API 请求时，控制台会显示接口返回的响应。在控制台中，你可以查看接口返回数据、Cookie、Header 和实际请求等详细信息。了解更多关于 Apifox 中的 [API 响应](https://docs.apifox.com/api-response.md)。

## Cookie 管理

网站利用 Cookie 来保存用户会话信息和个性化内容。Apifox 的 Cookie 管理器允许你查看不同域名的 Cookie，随 API 请求发送 Cookie，创建新的 Cookie，并通过代理或拦截器捕获 Cookie。了解更多关于[创建和发送 Cookie](https://docs.apifox.com/create-and-send-cookie.md)。

## 实际请求

送请求后，你可以在控制台功能查看并排查错误，同时查看接口实际发送的请求信息。了解更多关于[实际请求](https://docs.apifox.com/actual-request.md)。


## 提取响应示例

允许你将快捷请求保存为接口，以便在未来的测试场景中快速访问和引用，也可以将响应提取到 “响应定义” 或 “响应示例” 中。了解更多关于[提取响应示例](https://docs.apifox.com/extract-response-example.md)。

## 响应数据可视化

可视化面板使你能够输出图表，帮助你更有效地解读和分析响应数据。了解更多关于[响应数据可视化](https://docs.apifox.com/response-data-visualization.md)。
