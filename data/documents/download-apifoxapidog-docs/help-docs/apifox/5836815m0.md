# Web 端与客户端数据不同步如何处理？

若发现客户端中某个项目的接口数据与 Web 端不一致，那么有可能是因为两端数据未同步。你可以尝试以下两种方法解决：  

1. 退出 Apifox 客户端后重新运行。  

2. 进入 Apifox 客户端中的项目后，点击右上角的 “刷新” 按钮。
