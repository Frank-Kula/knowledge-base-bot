# 可以给测试场景下的请求添加统一的前后置操作吗？

Apifox支持设置测试场景的前后置操作。当你配置好场景前后置，运行该场景时，其下所有接口都会执行相同的操作。若想启用此功能，只需进入项目设置，打开场景前后置开关。

<Background>
![image.png](https://api.apifox.com/api/v1/projects/5097254/resources/524077/image-preview)
</Background>
开关开启之后，在场景详情页面就能添加前后置操作。
