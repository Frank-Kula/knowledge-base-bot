# 开放接口，选择导入 「OpenAPI/Swagger 格式数据 - URL 导入」，使用本地的服务地址无法导入

### 场景描述
- 开放接口的示例，URL 链接导入成功，项目权限及鉴权信息正确
- 该链接在前端页面可以进行导入，开放接口无法导入


### 问题原因

使用开放接口导入是在 Apifox 服务端执行的，用户的 URL 必须是外网可以访问的

### 解决方案
1. 链接在前端可导入，证明链接内的文件结构可识别，所以可以切换到字符串方式导入，即文件内容在去掉回车转成字符串
2. 该文件导入到 Github 的公开库中，该文件的链接即为外网可访问地址，可以使用 URL 导入的方式
