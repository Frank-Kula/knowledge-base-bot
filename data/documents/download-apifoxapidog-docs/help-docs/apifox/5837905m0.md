# 怎么给接口批量添加或删除 URL 前缀？

Apifox 目前不支持通过批量操作来修改接口 URL。不过你可以这样做：


<Steps>
  <Step>
    先从 “项目设置 -> 导出数据” 里把文件导出成 Apifox 格式。
      
<Background>

![CleanShot 2025-01-22 at 12.18.49@2x.png](https://api.apifox.com/api/v1/projects/5097254/resources/492167/image-preview)
</Background>

  </Step>
  <Step>
    然后用文本编辑器 *（比如 VSCode）* 批量查找并进行替换。
  </Step>
  <Step>
    最后再把修改好的文件导回 Apifox，这样就能达到批量修改的效果了。
  </Step>
</Steps>
