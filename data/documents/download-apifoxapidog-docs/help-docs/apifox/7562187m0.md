# 如何在自定义脚本中获取变量时，同时替换该变量的变量值中的变量或动态值

:::note[前置]
1.变量的变量值，引用了其他变量
2.变量的变量值，设置了动态值

基于以上两种场景，直接通过get的方式进行获取时，无法获取到变量值内引用的变量和动态值生成信息
```js
pm.globals.get("aa")
pm.environment.get("aa")
pm.variables.get("aa")
```
:::

```js
//自定义脚本使用语法可以直接替换变量内引用的变量和动态值
env_a=await pm.variables.replaceInAsync('{{env_a}}');
console.log(env_a)

env_c=await pm.variables.replaceInAsync('{{env_c}}');
console.log(env_c)

env_d=await pm.variables.replaceInAsync('{{env_d}}');
console.log(env_d)

```
