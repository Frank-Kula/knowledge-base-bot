# 常用字段

在 API 开发中，保持各个接口的一致性非常重要。Apifox 提供了 “常用字段” 功能，可以让具有相同名称和描述的字段在不同的接口间重复使用。

## 配置常用字段

要修改或设置你的常用字段，请按照以下步骤进行：

<Steps>
  <Step title="进入字段设置">
    点击 “修改接口” 界面中的 “说明” 部分的菜单按钮。
      
    <Frame>

    ![image.png](https://api.apifox.com/api/v1/projects/5097254/resources/476106/image-preview)
    </Frame>


      
:::highlight purple
或者，你也可以在 `项目设置` - `项目资源` - `常用字段` 中管理。
:::
  </Step>
  <Step title="输入详细信息">
    输入字段名称、描述以及其他必要信息。

<Frame>

![image.png](https://api.apifox.com/api/v1/projects/5097254/resources/510898/image-preview)
</Frame>

  </Step>
  <Step title="保存字段">
    确认你的输入，并点击“保存”按钮。
  </Step>
</Steps>

## 导入 / 批量编辑

支持使用英文逗号或冒号作为分隔符来导入或批量编辑常用字段。

## 使用常用字段

在接口中输入字段名称时，系统会自动显示预设的 “常用字段” 供选择。选择常用字段后，其描述将会自动填入。
