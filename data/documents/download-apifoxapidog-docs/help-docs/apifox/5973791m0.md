# 如何在 URL 路径中使用变量？

在 Apifox 中，路径中的变量部分我们通常会定义为一个路径参数。例如输入`{petId}`，可以在下方看到 Path 参数。
此时可以定义参数值为引用全局变量/环境变量值，例如 `{{id}}`。设置后当你发送请求时，实际请求中路径会被替换成变量 id 的实际值。
