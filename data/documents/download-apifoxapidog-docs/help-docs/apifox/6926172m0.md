# 为什么使用 Web 网页端，通过浏览器插件请求 Ollama 部署的大模型会提示 403？

### 原因
报错 403 是由于 Chrome 插件发送的请求携带 Header 参数 `origin: chrome-extension://xxx` ，而 Ollama 不支持这个头，导致请求被拒绝。

### 解决办法

需修改 Ollama 配置以放行 chrome-extension 来源的请求。具体可参照[官方 Issue #2335](https://github.com/ollama/ollama/issues/2335)，通过配置文件或启动参数添加白名单，确保含该头部的请求能正常访问。
