# Socket 如何使用 GBK 格式发送数据？

定位到 “项目设置 - 项目资源 - 自定义函数(Socket) - Socket 请求数据处理函数”，添加一个这样的函数:


<Background>
![image.png](https://api.apifox.com/api/v1/projects/5097254/resources/577360/image-preview)
</Background>


```js
var iconv = require("iconv-lite")
var encoded = iconv.encode(data, "GBK")
return encoded
```
然后修改 Socket 接口，添加这个数据处理函数即可。
