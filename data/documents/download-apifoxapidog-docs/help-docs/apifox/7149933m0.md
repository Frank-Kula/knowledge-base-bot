# 如何查看最近创建 / 修改的接口？

您可以通过以下两种方式掌握接口动态：  
#### 1. 主动查看近期变更
   - 进入系统后，点击左侧导航栏的接口根目录，选择 **全部接口**，进入接口列表页面。  
   - 从表头的设置中，将`创建时间`、`修改时间`的显示开关开启
   <Background>
![image.png](https://api.apifox.com/api/v1/projects/5097254/resources/550977/image-preview)
</Background>
   - 在接口列表中，可以看到接口的`创建时间` 和 `修改时间` 列，点击表头即可修改排序，直观查看近期创建或修改的接口。    
<Background>
![image.png](https://api.apifox.com/api/v1/projects/5097254/resources/550983/image-preview)
</Background>



#### 2. 开启变更通知提醒
   系统支持设置[通知对象](https://docs.apifox.com/notification-targets.md)，您可以联系管理员开启相关通知（如站内信、邮件提醒等），后续接口有创建、修改等操作时，会自动向您推送通知，无需手动查询~  

两种方式结合使用，能更及时、全面地掌握接口动态。
