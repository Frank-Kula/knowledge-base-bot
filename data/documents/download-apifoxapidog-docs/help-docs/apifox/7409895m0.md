# 为什么直接使用 Apifox 生成的 curl 运行报 “URL using bad/illegal format or missing URL“ ？

原因：复制出来的 curl 没有自动编码，导致直接使用该 curl 会报错 “URL using bad/illegal format or missing URL”

解决办法：需要手动将 `query&path` 参数值进行编码：打开 “接口 -> Params -> query&path 参数”，点击参数值输入框右键 “EncodeURIComponent” ，再保存，接着复制 cURL 即可。
