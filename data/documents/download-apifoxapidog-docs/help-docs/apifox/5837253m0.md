# 如何导入接口数据到 Apifox?

<Steps>
  <Step>
进入“项目设置 -> 导入数据” 面板。

<Background>
![CleanShot 2024-09-20 at 16.31.17@2x.png](https://api.apifox.com/api/v1/projects/5097254/resources/466562/image-preview)
</Background>
  </Step>
  <Step>
选择你需要导入的数据格式。
  </Step>
  <Step>
按照指引填写必要的信息，并点击 “继续”。
  </Step>
  <Step>
预览数据并确认导入选项。
<Background>
      
![CleanShot 2024-09-20 at 16.33.51@2x.png](https://api.apifox.com/api/v1/projects/5097254/resources/466563/image-preview)
</Background>
:::highlight purple
了解更多[导入选项](https://docs.apifox.com/import-settings.md)。
:::
  </Step>
  <Step>
完成导入。
<Background>
<img src="https://api.apifox.com/api/v1/projects/5097254/resources/466565/image-preview" width="640px" />

</Background>
  </Step>
</Steps>
