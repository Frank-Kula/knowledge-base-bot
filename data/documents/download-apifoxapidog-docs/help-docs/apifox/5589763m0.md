# 电话（Phone）

生成与电话相关数据的模块。

<br />

**概述**

要获取电话号码，请使用[数字（Number）](https://docs.apifox.com/5589760m0.md)模块。许多语言环境都提供特定于国家/地区的格式。



## imei

生成 IMEI 号码。

**返回**: 字符串

**示例**

```js
{{$phone.imei}}  // '73-603797-653587-5'
```


## number

生成一个随机的电话号码。

**参数**

| 名称  | 类型                                     | 默认值  | 描述                                                                                                                                                                                                                                          |
| ----- | ---------------------------------------- | ------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| style | 'human' \| 'national' \| 'international' | `human` | 生成的电话号码的样式：- `'human'`：（默认）人工输入的电话号码，例如 `555-770-7727` 或 `555.770.7727 x1234`- `'national'`：标准化国家格式的电话号码，例如 `(555) 123-4567`。- `'international'`：E.123 国际格式的电话号码，例如 `+15551234567` |

**返回**: 字符串

**示例**

```js
{{$phone.number}} // '1-203-635-9321 x1495'

{{$phone.number(style='human')}} // '(343) 464-9991'

{{$phone.number(style='national')}}  // '(549) 994-3486'

{{$phone.number(style='international')}}  // '+14642309454'
```
