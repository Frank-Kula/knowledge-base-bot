# 黑客（Hacker）

生成黑客/IT 词汇和短语的模块。

<br />

**概述**

有针对不同词性的方法，例如 `{{$hacker.abbreviation}}`、`{{$hacker.adjective}}`、`{{$hacker.noun}}`、`{{$hacker.verb}}` 和 `{{$hacker.ingverb}}`。或者，`{{$hacker.phrase}}` 创建一个组合这些词的较长短语。



## creditCardNumber

返回一个随机的黑客/IT 缩写。

**返回**: 字符串

**示例**

```js
{{$hacker.abbreviation}}  // 'AGP'
```


## adjective

返回一个随机的黑客/IT 形容词。

**返回**: 字符串

**示例**

```js
{{$hacker.adjective}}  // '在线'
```


## ingverb

返回一个用于连续动作的随机黑客/IT 动词（例如：ing 后缀；例如 hacking）。

**返回**: 字符串

**示例**

```js
{{$hacker.ingverb}}  // '复制中'
```


## noun

返回一个随机的黑客/IT 名词。

**返回**: 字符串

**示例**

```js
{{$hacker.noun}}  // 'feed'
```


## phrase

生成一个随机的黑客/IT 短语。

**返回**: 字符串

**示例**

```js
{{$hacker.phrase}}  // '我们需要传输神经 SDD 端口！'
```


## verb

返回一个随机的黑客/IT 动词。

**返回**: 字符串

**示例**

```js
{{$hacker.verb}}  // '重启'
```
