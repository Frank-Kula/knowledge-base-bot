# 为什么网络正常，访问 Apifox 提示“连接 Apifox 服务器出错，请检查网络是否异常”？

如果你的网络连接正常，但 Apifox 仍然提示“连接 Apifox 服务器出错”，可能是以下原因导致的：
1. 检查 Apifox 的网络代理设置： Apifox 的网络代理设置可能会干扰与服务器的连接。请检查你的 Apifox 代理设置是否正确，并根据需要进行调整。详情请查看[网络代理](https://docs.apifox.com/network-proxy.md)文档。
2. 浏览器访问 `apifox.com` 是否正常： 使用浏览器访问 `apifox.com`，如果无法访问，则可能是 Apifox 服务器出现问题或你的网络存在 DNS 解析问题。
3. 尝试更换网络： 尝试使用其他网络连接 Apifox，例如手机热点，以排除当前网络环境的问题。
4. 检查控制台报错信息： 打开 Apifox 控制台，查看 Network 标签页中的报错信息，这有助于确定问题的具体原因。如果你使用的是公司网络，某些网络策略可能会阻止 Apifox 的连接。请联系您的网管部门，将`*.apifox.com` 和  `*.apifox.cn` 添加到白名单。
5. 请提供一下 IP 地址，你可以通过 https://ip138.com/  查询。我们针对 IP 有风险拦截机制，若该 IP 所在区域有用户发起扫描行为，触发风控后可能会被拦截。

:::info[]
如何打开控制台：Windows：按 `alt+7+8`，macOS：按 `option + cmd + i`
:::
