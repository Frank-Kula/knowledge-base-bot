# 标签页使用说明（单击临时打开与双击固定打开）

在 Apifox 中，**点击目录树中的任何元素，都会在右侧打开一个 [标签页](https://docs.apifox.com/page-layout#%E6%A0%87%E7%AD%BE%E9%A1%B5)**。

### 标签页类型

标签页可能对应以下内容类型：

* 接口
* 数据模型
* 响应组件
* Markdown
* 目录
* 快捷请求
* WebSocket
* 项目概览
* 新建…



### 单击与双击的区别

Apifox 的标签页分为「**临时标签页**」和「**固定标签页**」，用于适配不同使用场景。

#### 单击打开（临时标签页）

* 单击目录树中的元素时，会在**临时标签页**中打开
* 标签标题显示为**斜体**
* 此时如果继续点击目录树中的其他元素，**当前标签页会被覆盖**
* 适合快速浏览、连续查看多个接口或文档

#### 编辑后自动固定

* 当你对标签页内容进行修改后
* 标签会自动变为**常规字体**
* 后续点击目录树中的其他元素时，将会**新建标签页**而不是覆盖当前标签
* 适合编辑、调试等需要保留上下文的场景

#### 双击打开（固定标签页）

* 双击目录树中的元素，可直接以**固定标签页**方式打开
* 标签标题立即显示为**常规字体**
* 不会被后续单击操作覆盖
* 适合需要长时间查看或对比的接口和文档
