# 导入 Markdown

Apifox 支持批量导入带 `.md` 后缀的 `Markdown` 文件，每个文件都会作为一个独立的 Markdown 文档被导入。



在 Apifox 中打开 “项目设置” 面板，点击 “导入数据”，选择 Markdown 格式的文件并上传。
