# 如何在持续集成中选择多个测试场景一起运行？

Apifox 支持同时运行多个测试场景。你可以通过以下两种方式实现：

1. 新建一个场景，引用这些场景： 你可以创建一个新的测试场景，并将需要运行的多个场景作为测试步骤添加到其中。这样，你就可以通过运行这一个新的场景来执行所有引用的场景。

<Background>

![image.png](https://api.apifox.com/api/v1/projects/5097254/resources/532594/image-preview)
</Background>


2. 使用 `-f` 命令运行指定目录： 你可以将多个测试场景放在同一个目录下，然后使用 Apifox CLI 的 `-f` 命令，结合目录 ID 来运行该目录下的所有测试场景。

:::info[]
- 更多命令请查看 [CLI 命令选项](https://docs.apifox.com/cli-command-options.md) 文档
- 目录 ID 在 “目录设置” 中可查看
- 根目录不支持指定
:::
