# 导入 .har 文件

`.har` 文件代表“HTTP Archive format”，是一种 JSON 格式的文件，用于记录浏览器与网站的交互过程。

`.har` 文件包含了网页请求和响应的记录，包括 URL、页面资源、HTTP 头信息、HTTP/HTTPS 请求和响应，以及浏览器从服务器发送或接收的其他数据。

## 导出 `.har` 文件

右键点击你要检查的网站，然后选择 “Network”，点击红色按钮开始记录 HTTP 请求和响应。
<Background>

![image.png](https://api.apifox.com/api/v1/projects/5097254/resources/468269/image-preview)
</Background>
选择你想要导出的 HTTP 请求和响应，然后点击 "Save all as HAR with content" 按钮。
<Background>

![image.png](https://api.apifox.com/api/v1/projects/5097254/resources/468270/image-preview)
</Background>

## 导入 `.har` 文件

打开项目的 “项目设置”，点击 “导入数据”，然后上传 `.har` 文件。
