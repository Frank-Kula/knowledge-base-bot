# 导入 Eolink

支持导入 Eolink 项目的 `.json` 格式文件。

## 导出 Eolink 数据

打开 Eolink 里对应项目，点击左侧菜单中的 “项目管理”，展开后前往 “项目设置”。

<Background>

![image.png](https://api.apifox.com/api/v1/projects/5097254/resources/468353/image-preview)
</Background>


展开后再点击「导出项目」按钮，选择「Eolinker 项目数据 (.json)」。


<Background>

![image.png](https://api.apifox.com/api/v1/projects/5097254/resources/468354/image-preview)
</Background>


:::tip[]
注意：在 Eolink 中，只有管理员才能导出数据。若无法导出数据时，请检查项目权限。
:::


## 导入 Eolink 数据

打开 Apifox 中的 “项目设置” 面板，点击 “导入数据 -> Eolink”，上传已导出的文件。
