# 科学（Science）

生成科学相关条目的模块。

<br />

**概述**

此模块中的两种方法都返回对象而不是字符串。例如，你可以使用 `{{$science.chemicalElementName}}` 挑选出你需要的特定属性。


## chemicalElement

返回一个随机的元素周期表元素。

:::info[]
原始的 `faker.science.chemicalElement()` 方法已被细化为三个更精确的变量：

*  `{{$science.chemicalElementSymbol}}`：生成元素符号。
*  `{{$science.chemicalElementName}}`：生成元素名称。
*  `{{$science.chemicalElementAtomicNumber}}`：生成元素原子序数。
:::

**返回**: 字符串

**示例**

```js
{{$science.chemicalElementAtomicNumber}} // '72'

{{$science.chemicalElementName}}  // '镱'

{{$science.chemicalElementSymbol}}  // 'Fm'
```


## unit

返回一个随机的科学单位。

:::info[]
原始的 `faker.science.unit()` 方法已被细化为两个更精确的变量：

*  `{{$science.unitName}}`：生成单位名称。
*  `{{$science.unitSymbol}}`：生成单位符号。
:::

**返回**: 字符串

**示例**

```js
{{$science.unitName}} // '法拉'

{{$science.unitSymbol}} // 'lm'
```
