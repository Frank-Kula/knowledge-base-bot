# Apifox 闪退、白屏怎么办？

## 方法一

可以右键点击电脑桌面菜单栏中的 Apifox 图标，选择退出后重新启动 Apifox，即可正常使用。

## 方法二

若方法一无效，并且你能稳定复现这个问题，请把 Apifox 拖到终端里面，然后按回车键 *（Enter）*。

<Background>

![image.png](https://api.apifox.com/api/v1/projects/5097254/resources/494374/image-preview)
</Background>

再次打开 Apifox，下次闪退后把终端里面最后输出的内容复制下来，然后发给我们的[技术支持](https://docs.apifox.com/contact-us.md)以帮你解决。
