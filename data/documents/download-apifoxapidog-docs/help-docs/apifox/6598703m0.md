# Apifox 发布文档/快捷分享中，如何让使用者修改请求的 Base URL ？

在 Apifox 发布文档/快捷分享文档中，若想让文档使用者设置修改 Base URL，可通过设置变量的方式轻松实现。具体操作步骤如下：
1. 创建变量并引用变量
在 Apifox 项目中，进入某个「环境」页面，点击「添加变量」，输入变量名称（例如“baseUrl”）。在环境对应的`前置 URL` 中的 URL 输入框中，使用变量引用（例如 `{{baseUrl}}`）来代替具体的 URL 地址。这样，接口的 Base URL 就与创建的变量关联起来。
   <Background>
   ![image.png](https://api.apifox.com/api/v1/projects/5097254/resources/520672/image-preview)
   </Background>

2. 访问文档
当文档发布分享给使用者后，使用者可以在界面上看到引用变量的地方。当鼠标hovor到变量引用处时，会提示使用者去设置变量值
    <Background>
    ![image.png](https://api.apifox.com/api/v1/projects/5097254/resources/520666/image-preview)
     </Background>
 
3. 修改变量值
点击变量处，自动弹出变量设置窗口，使用者只需填入所需的实际值。
     <Background>
    ![image.png](https://api.apifox.com/api/v1/projects/5097254/resources/520670/image-preview)
     </Background>
     保存后，文档上显示的路径、实际接口请求都将自动应用更新后的变量值。
      <Background>
    ![image.png](https://api.apifox.com/api/v1/projects/5097254/resources/520676/image-preview)
       </Background>
    
通过这种变量设置的方式，能有效解决不同使用者需要自定义 Base URL 以访问各自实际数据的问题。
