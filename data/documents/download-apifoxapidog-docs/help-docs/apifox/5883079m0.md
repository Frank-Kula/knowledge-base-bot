# 如何在 Apifox 的卡片组件中链接到项目内的文档或接口？

可以使用以下方式在卡片组件内创建跳转链接。


![image.png](https://api.apifox.com/api/v1/projects/5097254/resources/513792/image-preview)


#### 如何快速获取文档、接口、目录路径 ？
点击 Apifox 中的 "插入项目内接口/文档"按钮，选中资源插入后，资源路径将会显示在页面上。
