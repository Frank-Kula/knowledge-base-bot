# 数据库（Company）

生成数据库相关条目的模块。

<br />

**概述**

传统的 relational 数据库表中的数据按具有特定类型的列进行组织 - `column()`、`type()`。数据库通常有一个 `engine()` 和一个用于排序的默认 `collation()`。

对于 NoSQL 数据库 MongoDB，`mongodbObjectId()` 提供随机 ID。



## collation

返回随机数据库排序规则。

**返回**: 字符串

**示例**

```js
{{$database.collation}}  // 'utf8_unicode_ci'
```


## column

返回随机数据库列名。

**返回**: 字符串

**示例**

```js
{{$database.column}}  // '电话'
```


## mongodbObjectId

返回 MongoDB ObjectId 字符串。

**返回**: 字符串

**示例**

```js
{{$database.mongodbObjectId}}  // '32488b5ddc1de4fe5f74eff5'
```


## type

返回随机数据库列类型。

**返回**: 字符串

**示例**

```js
{{$database.type}}  // '时间'
```
