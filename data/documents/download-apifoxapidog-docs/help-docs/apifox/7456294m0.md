# 如何在文件修改后，接口直接读取修改的文件（本地文件路径以变量存储）


:::tip[该问题需要解决两方面的问题]

1. 修改文件内容，不管是 csv 文件或者别的什么，Apifox 不支持直接修改文件内容，但是可以使用外部程序，帮助文档查看 [调用外部程序](https://docs.apifox.com/call-external-programs.md)。

2. 想要解放双手，避免手动上传，自动使用本地的文件，需要使用变量作为中间容器
