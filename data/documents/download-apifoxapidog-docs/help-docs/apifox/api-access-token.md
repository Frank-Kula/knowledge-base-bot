# API 访问令牌


你可以通过 API 访问令牌来调用 Apifox 的开放 API 来查看、修改和使用 Apifox 中的各类数据。想了解更多详情，可以访问 Apifox 的[开放 API 文档](https://apifox-openapi.apifox.cn/)。

本教程将指导你获取 Apifox 的开放 API 访问令牌，帮助你更好地使用 Apifox 的开放 API 服务。

## 创建新令牌

按照下面的步骤创建新令牌：


<Steps>
  <Step>
    将鼠标悬停在页面右上角的头像上，点击 “账号设置 -> API 访问令牌”。
  </Step>
  <Step>
    点击 “新建”。

  
    <Background>

    ![image.png](https://api.apifox.com/api/v1/projects/5097254/resources/485747/image-preview)
    </Background>

:::caution[]
这个令牌在 Apifox 中拥有和你账号一样的权限，可以访问相应的[团队和项目](https://docs.apifox.com/member-roles-and-permissions.md)。
:::
  </Step>
  <Step>
    输入令牌名称并选择有效期。
  </Step>
    <Step>
    点击 “保存并生成令牌” 来创建令牌。
        
    令牌只会显示一次，所以要将其复制并保存下来，离开该页面后无法再次查看令牌，只能重新创建。
        <Background>
<img width="560px" src="https://api.apifox.com/api/v1/projects/5097254/resources/485749/image-preview"/>
</Background>


:::tip[]
修改令牌的名称或有效期不会重新生成令牌。如果不小心弄丢了令牌，需要点击 “新建” 重新创建一个。
:::
  </Step>
</Steps>
