# 常见问题

<Accordion title="在使用 AI 功能的时候，发现功能不能用、与预期不符、生成质量差，怎么回事？" defaultOpen>
如果你发现在使用 AI 功能的时候，等待时间久、生成质量差、没办法使用，请首先确认选择使用的模型是具体哪一种。模型的能力是影响 Apifox AI 功能效果的核心因素。我们强烈建议使用各个供应商最新发布的中杯非思考模型， 例如：`DeepSeek-V3-0324`、`doubao-seed-1.6-250615`、`qwen-plus-2025-04-28` 等。

</Accordion>

<Accordion title="我能在个人级别配置 AI 供应商和 API Key，然后我自己一个人使用 AI 功能吗？" defaultOpen={true}>
目前不支持个人配置 AI 功能并且仅个人进行使用，都需要在团队 / 组织级配置。
</Accordion>

<Accordion title="我常用的模型供应商在列表中没有，能不能支持更多供应商？" defaultOpen={true}>
目前，如果你需要使用预设供应商列表之外的供应商和模型，可以使用“自定义 API 配置”功能来自己配置模型使用。
</Accordion>

<Accordion title="有没有模型用量查看功能？能不能查看到哪些用户使用了多少用量？" defaultOpen={true}>
目前还不支持在 Apifox 中查看用量，你可以在模型供应商处查看配置的 API Key 的使用用量。
</Accordion>

<Accordion title="未来还会有哪些功能支持 AI？我有 AI 功能的建议、我发现了相关 bug，该怎么办？" defaultOpen={true}>
关于 Apifox AI 功能的相关反馈、建议，请加入我们的[交流群](https://docs.apifox.com/contact-us.md)并在里面提出。感谢每一位朋友帮助我们将功能打磨到更好。
</Accordion>
