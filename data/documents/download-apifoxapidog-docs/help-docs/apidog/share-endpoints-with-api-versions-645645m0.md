# Share endpoints with API versions

In the `Quick Share` feature, you can create online share for endpoints specific to a particular API version.
<Background>
![Quick Share API Version Endpoints](https://assets.apidog.com/help/assets/images/share-api-version-01-b2a967f4a8a68b33164ad41fa69e628e.png)
</Background>
When creating a new share, you can select the scope of endpoints for the quick share. You can choose a specific API version and then select all or some of the endpoints from this API version to generate an online share link.

<Background>
![Quick Share API Version Endpoints](https://assets.apidog.com/help/assets/images/share-api-version-02-3ed3a2e2523b5a35554e4f1ba31a1058.png)
</Background>
Users accessing the online share link will be able to view the content you want to share in specific API version.
<Background>
![Quick Share API Version Endpoints](https://assets.apidog.com/help/assets/images/share-api-version-03-29d2f7c03b5f9480785012847abe306d.png)
</Background>
