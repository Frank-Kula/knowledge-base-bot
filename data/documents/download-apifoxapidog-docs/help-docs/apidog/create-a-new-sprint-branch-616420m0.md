# Create a new sprint branch

When starting a new sprint with defined requirements, move to the technical design phase by accessing the relevant project at Apidog. Use the sprint branch switch feature next to the `APIs` and click on `New Sprint Branch` to create a new sprint branch and begin related tasks.

![Apidog Creating a New Sprint Branch](https://assets.apidog.com/help/assets/images/create-branch-01-dc945e095806ebd5713c6dc640419be8.png)

You can also click on the `Manage Sprint Branches` to view a list of sprint branches. Alternatively, go to `Settings` -> `Manage Sprint Branches` to see more details of existing sprint branches or choose to create a new sprint branch.

![Apidog Creating a New Sprint Branch](https://assets.apidog.com/help/assets/images/create-branch-02-5193480c8838bbd2eb73484c43635646.png)
