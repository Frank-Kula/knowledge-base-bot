# Where is Apidog's data stored, and how is data security ensured?

Apidog offers three different services, each with distinct data storage locations:

1. **Apidog SaaS Global:** Data for this service is stored in the United States, utilizing AWS (Amazon Web Services) infrastructure.
   
2. **Apidog SaaS Europe:** Data is stored in AWS located in Ireland to serve European clients with data residency requirements.

3. **Apidog On-premises:** All data resides on your own self-hosted servers and will not be stored on Apidog's cloud. No data will be sent to Apidog's server, ensuring complete control and privacy over your data.

Each of these storage options has been selected to ensure optimal security and compliance with regional data protection standards. Apidog employs various security measures to safeguard your data, which you can learn more about in our [security measures](https://legal.apidog.com/security-measures-645653m0).
